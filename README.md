# -B-J-Biscuit-Business-Analysis-Dashboard
 B&amp;J Biscuit Business Analysis Dashboard by using Power BI

## Project Overview  
This Power BI dashboard transforms raw sales data into interactive visual reports to allow quick evaluation of sales performance, product trends, and profitability.

## Data Sources & Structure  
- Data file: `sales_data.csv`  
- key columns: Date, Product, Category, Region, UnitsSold, UnitPrice, Revenue, Cost, Profit  

## Tools Used  
- Power BI Desktop  
- Power Query & DAX (for data transformation & metrics)
